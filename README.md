# SDL3

A Simple SDL3 Project that show a window

# Dependencies

You need [make](https://www.gnu.org/software/make/) and [cmake](https://cmake.org/) to build this project, you also need to install all the SDL3 dependencies for your environment
\
Please check directly [here](https://github.com/libsdl-org/SDL/blob/main/INSTALL.md)

# Build and Run

All the available commands for building and running the project
## Installation

Before all, install your dependencies

```bash
make install
```

## Build

Now you can build your executable with

```bash
make build
```

## Run

to execute your program, use

```bash
make run
```

## Clean

To clean your project directory, call

```bash
make clean
```
